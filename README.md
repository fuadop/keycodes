<h1 align="center">Keycodes</h1>

<p align="center">Access key codes with their respective key names.</p>

## Installation

- __yarn__
```sh
yarn add keycodes
```

- __npm__
```sh
npm install keycodes
```

## Usage
- __cdn__

```html
<script src=""></script>
```

- __modules__
```js
import keys from "keycodes"

const A = keys.A 
//this would return the respective keycode for the letter A(a).
```
